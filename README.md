# Filmify

Filmify is an Android application that allows users to capture and process photos using various filters and effects.

## Features

- Capture photos using the device's camera.
- Apply black and white filter to photos.
- Toggle between different blur effects.
- Zoom in and out while taking photos.
- Save processed photos to the device's gallery.
